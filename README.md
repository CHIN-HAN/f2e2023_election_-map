<h2 id="Intro">作品說明</h2>

<p>來自平行時空的貓貓國在 2020 年進行總統大選，這個即時開票地圖紀錄了喵夫人、橘博士和喵先生在 2020 年大選的政見以及投票結果。透過首頁的候選人介紹，可以得知該年度由哪隻貓貓當選、以及他們的政見。進入開票地圖，可以與地圖進行互動，透過地圖以及統計圖表得知全台、縣市以及村里的各政黨得票分佈。</p>
